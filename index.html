Archer Discord Bot Changelog

This is a full documentation of all the changes, updates, and bug fixes of the Archer Discord Bot by
hAck_3r64 (me!), also including every command the bot has as well as information on that specific
update.

The first version of the very early beta.
Its purpose, as requested by VimtoBlue#0118, was to display updates from the weekly newsletter.
It seemed wasteful to me to just leave it at that so I added a bunch of variety commands for fun,
which probably barely kept the server on its feet.

V0.1.1 (Beta)

14/02/2022

The first beta update.
Some bugs have been fixed:
- a!random facts not being randomised
- a!chatmode not toggling chatmode
- emojis not displaying
- pings not pinging

a!info has been updated to display the current version and the changelog website.
Three new commands have been added, too.

Commands

- a!members
Displays the amount of members in the server and how many are currently online.

- a!changelog
Another plug to the changelog website.

- a!randomvid
Generates a random video from my YouTube recommended (also pings me so I hope it isn't too overused).

V0.2.0 (beta)

17/02/2022

A massive update.
This update was a ground re-write of the bot's programming in order to increase bot stability, uptime, and
the speed that messages send. Unfortunately, some sacrifices had to be made and some commands had
to be removed to account for these stability updates. However, they will hopefully be added back in later
updates (as well as some new, fun commands!).
The commands removed were:
- a!time
- a!members
- a!randomvid
- a!chatmode
Bug fixes and changes in this update were:
- a!swear has been renamed to a!curse
- a!info has been updated to the current version and some grammatical changes have taken place
- Links now embed consistently

V0.2.1 (Beta)

18/02/2022

A nice, mini update.
A few bugs were fixed:
- Incorrect links
- Misspellings
- Messages failing to send
- Incorrect formatting
A total of five new commands have been added.

Commands

- a!quote
Replies with a random quote from a list of 255 quotes.

- a!qinfo
Shows links to sources for a!quote and links a form for changing or censoring a quote.

- a!short
Links a random YouTube Short from a list of 110.

- a!servers
Generates invite links to three different, very cool servers (The links don't expire!)

- There is also one secret command. One that I will not tell you. A command that may or may not be developed in the future...

V1.0.0

11/03/2022

The first version that wasn't part of the beta.
The bot is now permanently online!
Some changes were made:
- a!servers now generates an invite link to a random server out of a list
- The bot has a custom status now and is idle instead of online
A total of seven new commands have been added.

Commands

- a!miningaway
Posts all of the lyrics to 'Mine Diamonds'.

- a!egg
Makes the bot reply with 'egg'.

- a!bestgames
Shows the list of hAck_3r64's top 5 favourite games.

- a!commandslist
Shows a list of all* of the Archer Discord Bot's commands.

- a!ua
Makes the bot say '#weStandForUkraine :flag_ua:'.

- a!russellhoward
Links a random TikTok clip by the greatest comedian.

- a!clips
Links a random YouTube clip.

*a!commandslist has a chance of not displaying all commands, results may vary.
